# Shopify プラン比較（日本拠点向け）

## 目的

Shopify の3プランを日本拠点の前提で比較し、初期立ち上げに最適なプランを選定する。
過剰投資を避けつつ、必要な機能が揃っているプランを推奨する。

---

## コスト試算（月商別・海外カード決済前提）

初期想定: 月商30〜80万円程度（eBay 月間利益100万円の一部を Shopify に振り分け）

海外顧客中心のため、海外カード手数料（3.9% / 3.85% / 3.8%）で試算する。

| 月商 | Basic 総コスト | Grow 総コスト | 差額/月 | 判定 |
|---|---|---|---|---|
| ¥300,000 | ¥11,700 + ¥4,850 = **¥16,550** | ¥11,550 + ¥13,500 = **¥25,050** | -¥8,500 | Basic が有利 |
| ¥500,000 | ¥19,500 + ¥4,850 = **¥24,350** | ¥19,250 + ¥13,500 = **¥32,750** | -¥8,400 | Basic が有利 |
| ¥1,000,000 | ¥39,000 + ¥4,850 = **¥43,850** | ¥38,500 + ¥13,500 = **¥52,000** | -¥8,150 | Basic が有利 |
| ¥2,000,000 | ¥78,000 + ¥4,850 = **¥82,850** | ¥77,000 + ¥13,500 = **¥90,500** | -¥7,650 | Basic が有利 |
| ¥5,000,000 | ¥195,000 + ¥4,850 = **¥199,850** | ¥192,500 + ¥13,500 = **¥206,000** | -¥6,150 | ほぼ同等 |

※ Shopify Payments 利用前提。PayPal 併用時は別途 PayPal 手数料がかかる。

**損益分岐:** 海外カード手数料率の差が 0.05% と小さいため、手数料差だけでは月商がいくら伸びても Basic のコスト優位が逆転しない。Grow への移行判断はコストではなく機能面で行う。

## 機能面の評価

### Basic で十分な機能
- オンラインストアの構築・運営
- 商品登録数無制限
- 割引コード・ギフトカード
- SSL 証明書
- Shopify Payments（日本で利用可能）
- 手動の注文作成
- 基本的な売上レポート
- 配送ラベル印刷

### Grow で追加される主な機能
- スタッフアカウント5名（Basic は2名）
- 標準レポート（売上分析がより詳細）
- 市場別の高度な国際販売機能（多通貨・多言語の細かい制御が強化される）

### Advanced で追加される主な機能
- カスタムレポート（自由な軸で分析可能）
- 関税・輸入税の自動計算（海外顧客の購入時に表示）
- 外部配送業者の料金自動計算
- スタッフアカウント15名

## Shopify Payments の日本拠点での利用

Shopify Payments は日本で利用可能。
ただし以下の対応が必要:

- **特定商取引法に基づく表記ページ** の設置が必須（日本の法律要件）
- **本人確認・事業情報の登録** が必要（審査あり）
- **対応通貨:** 日本円での受け取りが可能。顧客には現地通貨で表示できる
- **振込サイクル:** 日本の銀行口座へ自動振込。詳細な振込サイクルは契約時の Shopify Payments 条件を確認

## 推奨案

### 推奨: Basic プラン（¥4,850/月・月払い）で開始

**理由:**

1. **コスト面で月商規模を問わず Basic が有利** — 海外カード手数料率の差が 0.05% と極めて小さく、月額差（¥8,650）を手数料差で回収できない。純粋なコスト比較では Basic が常に有利

2. **初期立ち上げに必要な機能がすべて揃っている** — 商品登録無制限、Shopify Payments、割引コード、基本レポートなど、300〜500 SKU のテスト運営に不足する機能がない

3. **個人運営ならスタッフアカウント2名で十分** — オーナー1名 + 予備1名で初期は問題ない

4. **プランのアップグレードはいつでも即時可能** — Shopify は管理画面からプラン変更が即日反映される。売上拡大・多言語展開のタイミングで Grow に切り替えても遅くない

5. **月払いで開始し、事業継続を確認後に年払いに切り替える** — 年払い（¥3,650/月）にすると約25%節約できるが、初期は月払いでリスクを最小化する

### アップグレード判断基準

| 条件 | 移行先 |
|---|---|
| スタッフを3名以上追加する必要がある | Grow（¥13,500/月） |
| 多言語・多通貨の細かい制御が必要になった | Grow |
| 関税・輸入税の自動計算を顧客に表示したい | Advanced（¥58,500/月） |
| 配送料金の外部業者連携による自動計算が必要 | Advanced |
