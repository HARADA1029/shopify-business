# Shopify 向け価格設計方針

## 1. 問題の整理

### 現状の eBay 価格に含まれるもの

```
eBay 出品価格 (USD)
  = 仕入原価
  + 国際送料（EMS / eパケット等の実費ベース）
  + eBay 手数料（最終価格の約13%）
  + 利益マージン
  + 【一部商品のみ】米国向け想定関税の上乗せ
```

### 3つの問題

| # | 問題 | リスク |
|---|------|--------|
| 1 | **関税上乗せの不統一** | 商品ごとに「関税込み」「関税なし」が混在しており、一律の計算式で Shopify 価格を導出できない |
| 2 | **二重取りリスク** | Shopify の Duties & Import Taxes 機能で checkout 時に関税を徴収する場合、関税上乗せ済み商品はバイヤーから二重に取ることになる |
| 3 | **チャネル間の価格矛盾** | 同じ商品が eBay と Shopify で大きく異なる価格で出ていると、バイヤーの不信感につながる。特に Shopify の方が高い場合はクレームリスクがある |

### 前提: 関税の仕組み

| 配送方式 | 関税の支払い者 | 現在の運用 |
|----------|--------------|------------|
| **DDP** (Delivered Duty Paid) | セラーが事前に負担 | 使っていない |
| **DDU** (Delivered Duty Unpaid) | バイヤーが受取時に支払い | eBay で採用中 |
| **Shopify Duties 機能** | バイヤーが checkout 時に事前支払い | 未導入（検討中） |

**重要:** 米国を含むすべての仕向地で、関税・輸入税が発生しうる前提で設計する。
免税枠（de minimis）の有無や閾値は国・時期によって変わるため、
価格設計の前提には組み込まない。


## 2. Shopify 向け価格設計の基本方針

### 方針: 「Shopify 基準価格」は関税を含まないクリーンな価格にする

```
Shopify 基準価格 (USD)
  = 仕入原価
  + 利益マージン
  + Shopify 決済手数料（約2.9% + $0.30）
  ※ 関税・輸入税は含めない
  ※ 送料は別途（商品価格に含めない）
```

**理由:**

1. Shopify では Duties & Import Taxes を checkout で徴収する選択肢がある
   → 商品価格に関税を含めると二重取りになる
2. 価格に関税が入っていないほうが「安く見える」→ コンバージョン率に有利
3. 将来的に国ごとの価格設定（Shopify Markets）を使う場合にも、
   関税なしのベース価格があった方が柔軟に対応できる
4. eBay 手数料（約13%）と Shopify 手数料（約2.9%）の差額分を
   バイヤーに還元できる → 競争力のある価格が出せる


## 3. eBay 価格との違い

### 手数料構造の比較

| 項目 | eBay | Shopify |
|------|------|---------|
| 販売手数料 | 約13%（FVF + payment processing） | 0%（Basic プランでも取引手数料なし※） |
| 決済手数料 | FVF に含む | Shopify Payments: 2.9% + $0.30 |
| 月額固定費 | ストア契約料 $21.95〜 | $39〜$399/月 |
| 送料ラベル | eBay 割引あり | Shopify Shipping 割引あり |

※ Shopify Payments 利用時。外部決済の場合は追加手数料あり。

### 価格の位置づけ

```
eBay 価格     = ベース価格 + eBay手数料分 + [関税上乗せ（一部）]
Shopify 価格  = ベース価格 + Shopify手数料分
                ↓
差額 = eBay手数料(約13%) - Shopify手数料(約3%) ≒ 約10%
```

**つまり: eBay と同じ利益率を維持しても、Shopify 価格は約10%安くできる。**


## 4. 「Shopify 基準価格」の算出方法

### Step 1: eBay 価格からベース価格を逆算する

```
eBay ベース価格 = eBay出品価格 ÷ (1 + eBay手数料率)
                = eBay出品価格 ÷ 1.13
```

### Step 2: Shopify 基準価格を計算する

```
Shopify 基準価格 = eBay ベース価格 ÷ (1 - Shopify手数料率)
                 = eBay ベース価格 ÷ 0.971
```

### 簡易暫定ルール（統合）

```
Shopify 基準価格 ≒ eBay 出品価格 × 0.91
```

**注意: これは暫定の簡易計算式であり、以下の前提に立っている:**

- eBay 手数料率が約13%であること
- Shopify 決済手数料が約2.9%であること
- 対象商品に関税上乗せが含まれていないこと
- 送料は eBay・Shopify ともに商品価格に含まれていないこと

**前提が異なる商品（関税上乗せ済み・高額品）にはこの式を適用しない。**

| eBay 価格 | Shopify 基準価格（暫定） | 差額 |
|----------:|------------------------:|-----:|
| $100 | $91 | -$9 |
| $200 | $182 | -$18 |
| $300 | $273 | -$27 |
| $500 | $455 | -$45 |
| $1,000 | $910 | -$90 |


## 5. 商品分類ごとの扱い

### 分類 A: 関税上乗せなし（大多数）

```
Shopify 基準価格 = eBay 出品価格 × 0.91（暫定簡易ルール）
```

- 一括変換可能
- **初期移行ではこの分類を優先的に選ぶ**
- 端数処理で $X.99 に丸める

### 分類 B: 関税上乗せ済み（一部商品）

```
Shopify 基準価格 = 手動レビューで個別設定
```

- 上乗せ額が商品ごとに異なるため一律計算できない
- **初期移行からは除外する**
- 第二弾以降で1点ずつ価格を手動で見直す

### 分類 C: 高額品（$1,000 以上）

```
Shopify 基準価格 = 手動レビューで個別設定
```

- 母集団の約6%（383件）
- 高額品は関税額も大きく、バイヤーの価格感度も高い
- 0.91 の簡易ルールでは不十分な場合がある
- **初期移行の第二弾で対応。1点ずつ手動で価格レビュー**

### 母集団に対する分類の分布

| 分類 | 件数 | 割合 | 初期移行での扱い |
|------|-----:|-----:|:---:|
| A: 関税上乗せなし・$1,000 未満 | 推定 5,000+ | ~80%+ | **一括変換で移行** |
| B: 関税上乗せ済み | 不明（要手動特定） | 不明 | **初期移行から除外** |
| C: 高額品 $1,000+ | 383 | 6.2% | **第二弾で手動レビュー** |

※ 分類 B の件数はオーナーによる手動特定が必要。


## 6. 送料の扱い

### 方針: 商品価格と送料を分離する

| 項目 | eBay | Shopify（提案） |
|------|------|----------------|
| 送料表示 | Free Shipping（商品価格に含む） | **送料別途表示** |
| 送料計算 | なし | 重量 × 仕向地ベースのテーブル |

**理由:**
- 仕向地ごとに送料が大きく異なる（US: $15〜 / EU: $20〜 / AU: $18〜）
- 商品価格に送料を含めると、近距離のバイヤーに不利になる
- Shopify では送料テーブルを柔軟に設定できる

**初期は送料別途で開始する。送料無料は導入しない。**

理由:
- Shopify 価格はすでに eBay × 0.91 に調整済み。さらに送料無料にすると利益が崩れる
- 国際配送で配送先・重量差が大きいため、一律送料無料は初期段階ではリスクが高い

条件付き送料無料の検討は、試験公開で送料表示と利益感を確認した後に行う。
導入する場合は、まず US 向けから検討する。


## 7. Shopify Duties & Import Taxes の扱い

### 選択肢

| オプション | 説明 | メリット | デメリット |
|-----------|------|---------|-----------|
| **A. 徴収しない（DDU）** | バイヤーが受取時に支払い | シンプル、現行 eBay と同じ | バイヤー体験が悪い（予想外の請求） |
| **B. checkout で徴収（DDP風）** | Shopify が計算して事前徴収 | バイヤー体験が良い、返品率低下 | Advanced プラン以上が必要（$299/月） |
| **C. 価格に含める** | 主要市場の平均関税を商品価格に上乗せ | 表示がシンプル | 二重取りリスク、国ごとに不公平 |

### 推奨: 初期は A（DDU）→ データ蓄積後に B を検討

- Basic プラン（$39/月）で開始する場合、Duties 機能は使えない
- まず DDU で運用し、顧客からの問い合わせ頻度を見て判断
- 商品ページに「Import duties and taxes are not included」を明記する


## 8. 初期移行候補への適用ルール

### 価格再設計ルール（初期移行 300〜500 SKU 向け）

```
対象:   分類 A（関税上乗せなし・$1,000 未満）のみ
ルール: Shopify 基準価格 = eBay 出品価格 × 0.91（暫定簡易ルール）
端数:   $X.99 に丸める
除外:   分類 B（関税上乗せ済み）、分類 C（$1,000 以上）
```

**このルールは暫定であり、以下のタイミングで見直す:**
- 初期移行の売上データが蓄積された時点
- eBay 手数料率に変更があった時点
- Shopify プランをアップグレードした時点

### 初期移行の価格設定フロー

```
1. 関税上乗せ済み商品を手動で特定し、初期移行から除外
     ↓
2. $1,000 以上の高額品を除外
     ↓
3. 残りの商品に Shopify 基準価格 = eBay 価格 × 0.91 を一括計算
     ↓
4. 端数処理（$X.99）
     ↓
5. Compare Price に eBay 価格を設定（「他サイトより安い」の訴求）
```

### Shopify の価格フィールドの使い方

| フィールド | 用途 | 値 |
|-----------|------|-----|
| **Price** | 実売価格 | Shopify 基準価格 |
| **Compare at price** | 取消線価格（元値表示） | eBay 価格 or メーカー希望小売価格 |
| **Cost per item** | 仕入原価（非公開） | 仕入原価 |


## 9. オーナーが決めるべきこと

### 必須（初期移行前に決定が必要）

| # | 決定事項 | 選択肢 | 推奨 |
|---|---------|--------|------|
| 1 | **関税上乗せ商品の特定方法** | (a) 商品ごとにフラグを手動で付ける / (b) 価格帯で一律に推定 / (c) 初期移行からは除外 | **(c) 初期移行から除外** |
| 2 | **Shopify 価格の割引率** | (a) eBay × 0.91（手数料差のみ還元） / (b) eBay × 0.88（やや積極的） / (c) eBay × 0.85（攻めた価格） | **(a) × 0.91 で開始** |
| 3 | **送料の扱い** | (a) 全品送料別 / (b) $X 以上送料無料 / (c) 全品送料込み | **(a) 全品送料別で開始** |
| 4 | **Duties の扱い** | (a) DDU（バイヤー負担） / (b) DDP（セラー負担） / (c) Shopify Duties で事前徴収 | **(a) DDU で開始** |

### 任意（運用開始後に調整可能）

| # | 決定事項 | 補足 |
|---|---------|------|
| 5 | Compare at price に eBay 価格を表示するか | 「他サイトより安い」の訴求になるが、eBay バイヤーが比較して eBay 離れする可能性も |
| 6 | Free Shipping の閾値 | $200? $300? データを見てから |
| 7 | 為替レート更新の頻度 | 仕入が円建ての場合、円安/円高で利益率が変動する |
| 8 | セール・クーポン戦略 | Shopify の方が eBay より柔軟にプロモーション可能 |


## 10. まとめ

### 修正版の要点

1. **すべての仕向地で関税が発生しうる前提**で価格を設計する
2. Shopify 基準価格は **関税を含まないクリーンな価格** とする
3. `eBay × 0.91` は **暫定の簡易ルール** であり、適用対象は「関税上乗せなし・$1,000 未満」の商品に限る
4. **関税上乗せ済み商品と高額品は手動レビュー前提** とし、初期移行からは除外する

### 初期移行商品に適用する価格ルール

```
対象: 関税上乗せなし かつ $1,000 未満の商品
計算: Shopify 基準価格 = eBay 出品価格 × 0.91
端数: $X.99 に丸める
送料: 別途（商品価格に含めない）
関税: DDU（バイヤー受取時負担）
```

### オーナーが最終決定すべきこと

1. **関税上乗せ済み商品をどう特定するか**（手動フラグ or 初期移行から一律除外）
2. **暫定割引率 0.91 で開始してよいか**（利益率との兼ね合い）
3. **DDU で開始してよいか**（バイヤーの関税負担を許容するか）
4. **Compare at price に eBay 価格を入れるか**（二重価格表示の是非）
