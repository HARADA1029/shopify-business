# 未決事項・意思決定トラッカー

このファイルはプロジェクト全体の未決事項を一元管理する。
各項目には「仮説」を置いて設計を進めるが、オーナー確認後に確定とする。

## ステータス定義

| ステータス | 意味 |
|---|---|
| OPEN | 未着手。仮説を置いて進行中 |
| NEEDS_INPUT | オーナーの判断待ち。設計側の材料は揃っている |
| DECIDED | 確定済み。日付と決定内容を記録 |
| DEFERRED | 今は決めない。再検討時期を記録 |
